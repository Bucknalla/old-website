    CURRENT PROJECTS

    Investigation into Ternary Logic
    This project entails an investigation into the advantages/disadvantages of using ternary encoding for communications. The aim of the project is to determine if ternary encoding can provide benefits over binary implementations such as reducing the number of clock cycles needed to perform error correction as well as to do a comparision of CPU vs Dedicated Hardware error detection/correction.
    Github
    FPGA

    Design of 4-Bit Counter (VLSI)
    The task of this assignment was to follow the process of designing a 4 bit counter with syncronous reset logic, in CMOS. A report follows the complete process, from the design of the inverters to the simulation of the complete counter. Each step outlines the schematic in S-Edit, test schematic in S-Edit, the layout in L-Edit, Waveform Charts in W-Edit (Both S-Edit and L-Edit). The design suite used was Tanner Tools EDA.
